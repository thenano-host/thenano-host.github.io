## 🚀 Deployment auf GitHub Pages
1. Repository erstellen: `thenano-host.github.io`
2. Dateien (`index.html`, `404.html`, `README.md`) hochladen
3. Settings → Pages → Branch: `main` → Save
4. Fertig! 🎉

## ⚠️ Hinweis
Diese Webseite ist ein **statisches Frontend**. Für echte Minecraft-Server wird ein Backend auf einem VPS benötigt (z.B. mit Pterodactyl Panel).
